# Student Grade Calculator (C)

## 📌 Description
This is a simple C program that calculates a student's total marks, percentage, and grade based on marks entered by the user.

## ✨ Features
- Accepts student name
- Accepts marks for 3 subjects
- Calculates total marks
- Calculates percentage
- Displays grade

## 🛠️ Technologies Used
- C Programming
- GCC Compiler
- Visual Studio Code
- Git & GitHub

## ▶️ How to Run

Compile:

```bash
gcc grade_calculator.c -o grade_calculator
```

Run:

```bash
.\grade_calculator
```
